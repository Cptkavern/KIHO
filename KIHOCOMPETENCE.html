Manuel d'Utilisation KIHO

1. Dashboard (Effectifs)

C'est la tour de contrôle de votre filiale. Elle permet de structurer vos besoins.

  • Nouveau métier : Créez un profil de poste. Indiquez la Cible (nombre minimum d'autonomes requis) et la Périodicité (délai en mois pour les évaluations).
  • Nouveau collaborateur : Ajoutez un agent et affectez-lui ses métiers via le menu déroulant.

2. Référentiels (Contenu technique)

  • Configuration : Définissez les thèmes et les compétences.
  • Imprimante : Génère une grille d'évaluation papier vierge.
  • Import/Export JSON : Sauvegardez ou restaurez uniquement la structure technique d'un métier spécifique.

3. Évaluations (Saisie)

  • À ÉVALUER : Indique qu'une évaluation est requise (délai dépassé).
  • Notation : 0 = Non acquis | 1 = Accompagné | 2 = Sait faire | 3 = Sait former.

4. Gestion des données

KIHO fonctionne 100% en local. Exportez votre base régulièrement.

  • Exporter base globale : Sauvegarde TOUT le contenu de l'application dans un fichier.
  • Importer base globale : Restaure une sauvegarde complète.
