CONFIDENTIALITÉ & ENREGISTREMENT

IMPORTANT

Si vous actualisez la page ou fermez cet onglet, vos données non enregistrées seront perdues.

Pensez à enregistrer régulièrement votre travail via le bouton "Enregistrer" présent dans le menu situé en haut à droite.

Manuel d'Utilisation KIHO

1. Dashboard (Effectifs)

C'est la tour de contrôle de votre filiale. Elle permet de structurer vos besoins.

  • Nouveau métier : Créez un profil de poste. Indiquez la Cible et la Périodicité.
  • Nouveau collaborateur : Ajoutez un agent et affectez-lui ses métiers via le menu déroulant.

2. Gestion des données

KIHO fonctionne 100% en local. Enregistrez votre base régulièrement.

  • Enregistrer : Sauvegarde TOUT le contenu dans un fichier .json.
  • Ouvrir : Restaure une sauvegarde complète.
