Guide d'utilisation

Comment utiliser l'application KIHO

1. Filiale et Logo

Pour personnaliser vos documents, cliquez sur "NOM DE LA FILIALE" dans la barre supérieure pour taper le nom de votre réseau. Utilisez ensuite le menu (en haut à droite) et cliquez sur "Logo Filiale" pour charger l'image de votre choix.

2. Créer et Gérer les Rituels

Cliquez sur le bouton "Nouveau Rituel" (à gauche). Vous pouvez choisir de partir d'un modèle standard (Quotidien, Conducteur, Performance...) qui pré-remplira l'agenda, ou taper le nom d'un nouveau type de rituel personnalisé pour créer un modèle vierge.

3. Édition du Document

  • Cliquez simplement dans n'importe quelle case pour écrire.
  • Les majuscules : La première lettre se met en majuscule automatiquement pour plus de propreté.
  • Ajout de lignes : Utilisez les boutons "+ Ajouter" bleus pour insérer des objectifs, des entrées, des sorties ou des lignes à l'agenda.
  • Suppression de lignes : Survolez une ligne existante, un bouton de corbeille rouge apparaîtra sur la droite.
  • Sauvegarde automatique : Inutile de sauvegarder pendant la frappe, l'application enregistre tout en temps réel dans votre navigateur.

4. Cartographie

Ouvrez le menu en haut à droite et cliquez sur "Cartographie". L'application génère automatiquement un tableau récapitulatif de tous vos rituels, classés par Unité/Service et par fréquence. Chaque type de rituel possède un code couleur distinct.

Très Important : Sauvegarde de vos données

L'application fonctionne directement dans votre navigateur. Si vous videz le cache de votre navigateur, vous perdrez vos rituels.

Solution : Utilisez le bouton "Exporter" dans le menu pour télécharger un fichier (.json) contenant toute votre filiale sur votre PC. En cas de besoin, utilisez "Importer" pour recharger ce fichier.
